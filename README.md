# Abu Bakar - Portfolio Website

A modern, animation-rich portfolio website for Abu Bakar, a Marketing Strategist and Performance Marketer.

## Tech Stack

- **React** - UI framework
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations
- **React Router** - Navigation
- **Lucide React** - Icons
- **React Markdown** - Blog content rendering
- **date-fns** - Date formatting

## Features

- Responsive design optimized for all devices
- Smooth scroll-triggered animations
- Dynamic blog with category filtering
- Interactive case studies with modal details
- Contact form with validation
- Published research showcase
- Resume/CV page

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
src/
├── components/
│   ├── animations/    # Reusable animation components
│   ├── layout/        # Navigation, Footer, Layout
│   └── ui/            # UI components (buttons, etc.)
├── data/              # Static data (case studies, blog posts, etc.)
├── pages/             # Page components
└── hooks/             # Custom React hooks
```

## Pages

- **Home** - Hero, services preview, case studies, testimonials
- **About** - Bio, timeline, values
- **Services** - Detailed service offerings
- **Case Studies** - Project showcases with metrics
- **Research** - Academic publications
- **Resume** - Professional experience and skills
- **Blog** - Marketing insights and articles
- **Contact** - Contact form and FAQ
